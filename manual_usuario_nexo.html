Primeros Pasos
🚀
Configuración
Protocolo de Instalación
Nexo guía tu configuración mediante un proceso automatizado de 5 etapas para asegurar una sincronización perfecta.
01
Perfilado

Defines tu nombre y la temática de tu biblioteca (Literatura, Técnica, Investigación) para personalizar el Dashboard.

02
Motor Visual

El sistema detecta la presencia de Obsidian, requisito esencial para renderizar tu grafo de conocimiento.

03
Despliegue

Se crea automáticamente el Vault en Documents/Nexo y se instalan los plugins necesarios.

04
Energía IA

Vinculación de llaves API para el Ecosistema Híbrido de IA. El sistema se ilumina al estar listo.

Abre el archivo .env en la raíz de Nexo y reemplaza los valores:

GROQ_API_KEY=tu_llave
OPENROUTER_API_KEY=tu_llave
Uso Diario
🔄
Procesamiento
Ciclo de Trabajo
El pipeline de Nexo transforma tus notas en conocimiento estructurado en tres pasos clave.
1. Analizar

Generación de metadatos YAML con etiquetas, tono y temática en cada documento.

2. Enlazar

Inyección automática de Wikilinks que activan el grafo relacional dinámico.

3. Descubrir

Detección de patrones ocultos y generación de "Semillas de Ideas" que expanden tu biblioteca.

💬
Interacción
Chat Inteligente (RAG)
Pregúntale a tus propios documentos. Nexo utiliza Generación Aumentada por Recuperación (RAG) para responder usando tu propia información.
Caché de Inteligencia:

Si repites una consulta, Nexo entrega la respuesta instantáneamente sin consumo de tokens gracias al almacenamiento local de hallazgos semánticos.

🌐
Exploración
Perspectivas Visuales (MRV)
El Visor Web ofrece 4 dimensiones para navegar por tu universo de conocimiento.
  • 🎯 Radar: Dashboard de ideas proyectadas.
  • 🔷 Constelaciones: Agrupación por géneros.
  • 🌱 Mapa: Grafo concéntrico interactivo.
  • ◆ Hubs: Foco en temas y conexiones de red.

Control
Centro de Comando Superior

Modo Ocio (Idle)
